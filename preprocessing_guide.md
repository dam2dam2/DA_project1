# 데이터 전처리 스크립트 실행 가이드

이 문서는 `preprocessed_data.csv` 파일의 데이터를 보완하기 위한 스크립트 실행 방법을 안내합니다.

## 시스템 요구 사항
- Python 3.8 이상
- Pandas 라이브러리 (`pip install pandas`)

## 데이터 전처리 실행 방법 (통합 파이프라인)

기존의 분석용 칼럼 생성 및 전처리 과정을 자동화한 통합 파이프라인(`run_pipeline.py`)을 사용합니다. 이 스크립트는 원본 Excel 파일만으로 모든 분석 데이터를 동일하게 생성합니다.

1.  **실행 환경**: `/Users/dmjeong/innercircle/.venv`
2.  **실행 명령어**:
    ```bash
    /Users/dmjeong/innercircle/.venv/bin/python /Users/dmjeong/innercircle/DA_project1/run_pipeline.py
    ```
3.  **상세 로직**: `pipeline_guide.md`를 참고해 주세요.
   - 출력되는 로그를 통해 처리 결과를 확인할 수 있습니다.

## 주요 처리 로직
- **재구매 횟수**: 동일한 연락처로 서로 다른 날짜에 주문한 횟수를 카운트합니다. (총 고유 주문일수 - 1)
- **지역 정보**: 주소 텍스트 및 우편번호를 분석하여 '경기', '서울' 등의 정보를 `광역지역` 칼럼에 채웁니다.
- **정식 명칭 추가**: `광역지역` 정보를 바탕으로 '서울특별시', '경기도'와 같은 정식 명칭을 담은 `광역지역(정식)` 열을 새롭게 생성합니다.
